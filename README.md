# lingualink

Маркетплейс контента для изучения языков.

Платформа объединяет идеи **Duolingo** (практика, упражнения, прогресс) и **Udemy** (маркетплейс курсов от креаторов).

---

## Идея проекта

Пользователи могут:

* изучать языки через курсы и упражнения
* отслеживать свой прогресс
* общаться с другими пользователями
* практиковаться через чат с AI
* использовать встроенный переводчик
* читать книги по уровням языка

Креаторы могут публиковать свои курсы и зарабатывать рейтинг.
Администраторы модерируют контент и пользователей.

* создание курсов

## Структура курса

```
Course
 ├── Module
 │     ├── Lesson (HTML контент)
 │     └── Exercise
```

* структура фиксированная
* уроки могут содержать HTML-разметку
* упражнения могут включать тесты / ввод текста
